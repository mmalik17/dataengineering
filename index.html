CLAUDE AI ECONOMIC INDEX DATA PIPELINE

with Docker, dbt, Google Cloud Platform, and Streamlit dashboard

This project contains an end-to-end data engineering pipeline and interactive dashboard visualizing Claude AI's economic impact and usage patterns across the global economy. The project leverages Anthropic's Economic Index (AEI) dataset (survey taken date: Feb 5–12, 2026).

Objectives:
1. Map Global Adoption of Claude's usage share at both national and subnational (state/province) levels
2. Validate Productivity Gains by comparing the time taken for tasks with and without AI
3. Assess Skill Democratization by comparing human education requirements against AI-demonstrated knowledge

Challenge:
1. The data has large number of columns, so the data transformation is very crucial to filter out only the column we need
2. Connecting the local script, to Google Cloud Platform need several credential setup

NY TAXI DATA PIPELINE

with SparkSQL and Google BigQuery

In this project, the data pipeline is connecting the data from NYC Taxi website as main source, transforming the data using SparkSQL, and load the transformed data to BigQuery datamart. The datamart contains several business metrics to evaluate the trip data of NYC Taxi, to identify which location and route that has a biggest trip. For simplicity, the scope of the data range is only at November 2025

Objectives:
Create a data pipeline that transform the NY Taxi data source to data mart in BigQuery containing three business metrics, such as:
1. The daily trip counts
2. The most frequent pickup locations
3. The top routes (pickup-dropoff location pair)

Challenge:
1. Data source format is parquet file, so it need to be transformed before loading to data mart
2. There are outlier related to date. The data scope should only contain data from November 2025, but the pickup_date column shows data from 2025-10-26, 2025-10-27, 2025-10-30, and 2025-10-31
